HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to a user

CSS

  • CSS- cascading style sheets
  • The programming language used to define how a webpage conent in presents to users
  • This is where colors, aesthetics, visual layout, and organization are coded.
  • Internal Style Sheet contains CSS rules in the head section of the HTML file
  • Inline CSS works directly in the HTML file and applies to a specific HTML tag.
  • External CSS Style Sheet is the most common. This keeps CSS in a seperate file and links it using the "link" element.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently working on
  • git checkout -b branch name: creates a new branch and switches to it

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A function is a simple set of instructions that tells the computer how to perform a task
  • JS is case senstive!
  • An array is a group of related data shown with a varaible. (ex.shapes,student info)
  • An index is used to locate an item inside an array. This will typically start with item 0. (ex. shapes[0])
